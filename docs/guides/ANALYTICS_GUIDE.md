# Analytics Guide

This guide explains each analytics feature in the Fast6 dashboard, helping users interpret the data and make informed first touchdown picks.

---

## Analytics Page (Analysis tab)

The **Analysis** page (`/analytics`) is organized into three sections, each accessible via tabs:

### Overview tab

- **Your stats** — Win rate, Brier score, total picks, wins/losses, and streaks
- **ROI by week** — Return on investment over the season (line chart)

### Performance tab

- **Performance by picker** — Multi-user comparison: win rate, Brier score, picks, wins/losses, streaks
- **Player performance** — Top players by first touchdown frequency
- **Accuracy by odds range** — How well favorites vs longshots hit
- **Team matchup accuracy** — Accuracy when picking against each team

### Touchdowns tab

A single table with a **view toggle** and filters:

- **View:** "All touchdowns" or "First TD only"
- **Filters:** Week, team
- **All touchdowns** — Every touchdown scored in the season; "First TD" column shows Yes/— for each row
- **First TD only** — Only the first touchdown of each game; "First TD" column hidden

---

## Other Dashboard Sections

### Leaderboard

- Group standings, total picks, first TD wins, any time TD wins, points, win rate, average odds
- Points: 3 for First TD, 1 for Any Time TD

### Week Picks

- All picks for a selected week, by user, with odds and calculated returns

### Weekly Schedule

- NFL game schedule by week with results and Vegas lines

### Matchup Analysis

- Head-to-head team stats for any game; TD scorers when game is final

---

## Glossary

- **First TD** — The first touchdown scored in a game
- **Any Time TD** — Any touchdown scored by a picked player, regardless of order
- **Brier Score** — A measure of prediction accuracy (lower is better)
- **Implied Probability** — The chance of an event occurring, based on betting odds
- **ROI** — Return on investment; profit/loss vs amount risked
